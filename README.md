# art_horses
RedemRP Script for Horses Accessories Managment

## How to install

* Copy and paste ```art_horses``` folder to ```resources```
* Load ```art_horses/sql.sql``` file in your database
* Add ```start art_horses``` to your ```server.cfg``` file
* Have Fun!
